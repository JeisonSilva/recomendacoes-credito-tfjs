# Recomendacoes de Credito com TensorFlow.js

Projeto da faculdade para estudo de recomendacoes de credito usando JavaScript e TensorFlow.js.

## Requisitos
- Node.js 18+

## Instalar
```bash
npm install
```

## Executar exemplo
```bash
npm start
```

## Estrutura
- `src/index.js`: exemplo inicial de uso do TensorFlow.js
